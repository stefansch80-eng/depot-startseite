Diese Dateien koennen direkt kostenlos veroeffentlicht werden.

Empfohlene Optionen:
- GitHub Pages: publish-Ordner in ein oeffentliches Repository hochladen und Pages aktivieren
- Cloudflare Pages: publish-Ordner per Direct Upload hochladen

Wichtig:
- Vor jeder neuen Veroeffentlichung erst generate-app-data.ps1 oder direkt publish-site.ps1 ausfuehren
- Danach den aktualisierten publish-Ordner neu hochladen
